Example (Bouncing Ball)

공이 1야드 높이에서 떨어져서 이전 높이의 2/3만큼 튀어 오른다. 공이 움직인 총 거리는?


(공이 1 -> 2/3 -> 4/9 로 튀는 그림)
풀이 과정:
  • 내려갈 때: 1 (처음 낙하)
  • 튀어 오르고 내려갈 때: \(2 \times \frac{2}{3}\) (올라갈 때 2/3, 내려갈 때 2/3이므로 2배)
  • 두 번째 튀어 오를 때: \(2 \times (\frac{2}{3})^2 = 2 \times \frac{4}{9}\)
  • 전체 거리 식: \(1 + 2 \cdot \frac{2}{3} + 2 \cdot \frac{4}{9} + 2 \cdot \frac{8}{27} + \dots\)
  • \(= 1 + 2 (\frac{2}{3} + \frac{4}{9} + \frac{8}{27} + \dots)\)
  • 첫 번째 항 1을 제외하고 나머지 항들을 2로 묶어낸다.
  • 괄호 안의 급수는 첫째 항 \(a=2/3\), 공비 \(r=2/3\)인 기하 급수이다.
  • 합 \(S = \frac{2/3}{1-2/3} = \frac{2/3}{1/3} = 2\).
  • 따라서 총 거리는 \(1 + 2(2) = 5\)이다.
Example (Repeating Decimal)

순환소수 \(0.785714285714\dots\)를 분수로 바꾸기.

풀이 과정:
  • 숫자를 분석하면 \(0.5 + 0.285714285714\dots\)로 나눌 수 있다.
  • \(0.285714\dots\) 부분은 \(285714\)가 반복되므로, 첫째 항 \(a=0.285714\), 공비 \(r=10^{-6}\)인 급수이다.
  • 공식 \(S = \frac{a}{1-r}\)을 적용한다.
  • \(S = \frac{0.285714}{1-10^{-6}} = \frac{0.285714}{0.999999} = \frac{285714}{999999}\).
  • 약분하면 \(\frac{2}{7}\)이 된다.
  • 최종 결과: \(0.5 + \frac{2}{7} = \frac{1}{2} + \frac{2}{7} = \frac{11}{14}\).
Example 1

\(n \to \infty\)일 때 \(\frac{(2n-1)^4 + 1 + 9n^8}{1 - n^3 - 7n^4}\) 의 극한 구하기.

풀이 과정:
  • 분자와 분모의 최고차항을 찾는다. 분자는 \(n^4\) 차수이고, 분모도 \(n^4\) 차수이다.
  • 분모, 분자를 \(n^4\)으로 나눈다.
  • 분자: \((2n-1)^4/n^4 \to 2^4=16\). \(\quad (1+9n^8/n^4)\) (※ 원본 파일 표기 따름: \((1/n^8)+9 \to 9\) → \(\sqrt{9}=3\)으로 해석되는 문맥으로 보임) \(\to 3\).
  • 분모: \(1/n^4 - 1/n - 7 \to -7\).
  • 극한값: \(\frac{16+3}{-7} = -\frac{19}{7}\).
Example 2

\(\lim_{n \to \infty} \frac{\ln n}{n}\) 구하기.

풀이 과정:
  • \(n\)을 실수 \(x\)로 생각하고 로피탈의 정리(분모, 분자를 각각 미분)를 쓴다.
  • \(\ln n\) 미분 \(\to 1/n\), \(n\) 미분 \(\to 1\).
  • \(\lim \frac{1/n}{1} = 0\).
Example 3

\(\lim_{n \to \infty} (1/n)^{1/n}\) 구하기.

풀이 과정:
  • 바로 극한을 구하기 어려우므로 로그를 취한다.
  • \(y = (1/n)^{1/n}\)이라 하면
  • \(\ln y = \frac{1}{n} \ln(1/n) = -\frac{\ln n}{n}\).
  • Example 2에서 \(\frac{\ln n}{n} \to 0\)임을 구했다.
  • 따라서 \(\ln y \to 0\)이다.
  • 우리가 구하는 값은 \(y\)이므로 \(y = e^0 = 1\)이다.
Example 2 (Telescoping Series)

\(\sum_{n=2}^{\infty} \frac{2}{n^2-1}\) 의 합 구하기.

풀이 과정:
  • \(2n^2-1\)이 아니라 \(n^2-1 = (n-1)(n+1)\)이다.
  • 부분분수 분해 공식을 쓴다: \(\frac{1}{n-1} - \frac{1}{n+1}\).
  • \(n=2\)부터 대입하여 나열해 본다.
  • \(n=2: (\frac{1}{1} - \frac{1}{3})\)
  • \(n=3: (\frac{1}{2} - \frac{1}{4})\)
  • \(n=4: (\frac{1}{3} - \frac{1}{5})\) ... 앞의 \(-1/3\)과 뒤의 \(1/3\)이 지워진다.
  • 이런 식으로 계속 지워지면(telescoping), 맨 앞의 1과 1/2만 남고 뒤쪽 항들은 사라지거나 0으로 수렴한다.
  • \(S = 1 + \frac{1}{2} = \frac{3}{2}\).
Example 3

\(\sum \ln(\frac{n}{n+1})\) 조사하기.

풀이 과정:
  • 로그 성질을 이용해 분해한다: \(\ln n - \ln(n+1)\). (분의 1 꼴이 아니다)
  • 부분합 \(S_n = (\ln 1 - \ln 2) + (\ln 2 - \ln 3) + \dots + (\ln n - \ln(n+1))\).
  • 가운데 항들이 다 지워지고 \(\ln 1 - \ln(n+1) = -\ln(n+1)\)만 남는다.
  • \(n \to \infty\) 일 때 \(-\ln(n+1) \to -\infty\) 이므로 이 급수는 발산한다.
