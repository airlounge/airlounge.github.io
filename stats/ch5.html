대기과학 예제 4 (층간 기울기의 법선벡터)

연직 단면 상에서 온도장 T(x,z)를 근사하기 위해 두 점

를 3차원 공간 좌표로 본다면, 둘과 (0,0,0)이 이루는 평면은 “온도 기울기 방향”을 포함한다. a = r₁, b = r₂라 두고 n = a×b를 계산하면 평면의 법선벡터를 얻고, 이로부터 평면 방정식과 (x,z)에 대한 선형 근사가 가능하다. 이는 실제로는 다차원 회귀로 구현되지만, 기하학적 구조는 같다.

5.3 점–평면 거리

이다.

6. 고유값, 고유벡터, 대각화와 모드

6.1 정의

선형변환 M에 대해, 0이 아닌 벡터 r과 상수 λ가 

Mr = λr

을 만족하면 λ는 고유값, r은 고유벡터이다. [ppl-ai-file-upload.s3.amazonaws](https://ppl-ai-file-upload.s3.amazonaws.com/web/direct-files/attachments/65877188/0d1e6ef9-6b9a-4e9f-a968-2069f836cec7/Boas_3jang.pdf)

고유값은 특성방정식 

det(M - λI) = 0

의 해이다. [ppl-ai-file-upload.s3.amazonaws](https://ppl-ai-file-upload.s3.amazonaws.com/web/direct-files/attachments/65877188/0d1e6ef9-6b9a-4e9f-a968-2069f836cec7/Boas_3jang.pdf)

6.2 예제와 대각화

예제 9 (2×2 고유값 문제)

M = ( 5  -2
     -2   2 )

특성방정식: 

det( 5-λ  -2
     -2  2-λ ) = (5-λ)(2-λ) - 4 = λ² - 7λ + 6 = 0

→ λ = 1, 6

λ=1일 때: 

(5-1)x - 2y = 0 → 2x - y = 0 → y = 2x

λ=6일 때: 

(5-6)x - 2y = 0 → -x - 2y = 0 → y = -½x

        [ η₂ ]        [ -1  2 ] [ η₂ ]

( -1  2 )      [   ]
